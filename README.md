# bd-vorstellung

Kurze Selbstvorstellung von Dirk Siemers als reveal.js-Präsentation.

Die persönlichen Folien stammen aus dem Vortrag
[`sensor-to-smart-farm`](https://github.com/dirksiemers/sensor-to-smart-farm),
hier herausgelöst als eigenständige Vorstellungs-Präsentation.

## Aufrufen

`index.html` im Browser öffnen. Steuerung: Pfeiltasten / Leertaste,
`S` öffnet die Speaker-Notes, `F` Vollbild.

## Technik

- reveal.js 6.0.1 (lokal unter `dist/`, kein CDN nötig)
- Hintergrundbilder und Video unter `assets/`
